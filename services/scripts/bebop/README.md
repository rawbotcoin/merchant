# Rawbot - Drone


## Requirements
- Parrot Bebop 2 drone
- Joystick/Xbox controller (Mad Catz Fightpad used in this project)
- Joystick driver installed on your OS

## .env

```sh
NETWORK="ROPSTEN"
ACCOUNT_ADDRESS=""
ACCOUNT_PRIVATE_KEY=""
INFURA_KEY=""
RAWBOT_ADDRESS=""
DEVICE_MANAGER_ADDRESS=""
DEVICE_ADDRESS=""
MINIMUM_ALTITUDE=
MAXIMUM_ALTITUDE=
```

## Instructions - Use it on your own risk

```sh
$ npm install
$ npm start in
$ npm start out
```
